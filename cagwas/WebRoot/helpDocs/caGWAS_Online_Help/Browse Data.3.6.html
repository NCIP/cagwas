Searching SNP Subject Data
Only registered users may access data about subjects and controls.
To search by subject data
1.
Click the Browse Data menu. The Browse Data page appears.
2.
Select a study and a version. If you do not select a version, your search will return the most recent study data.
3.
Click the Subject Data dataset option.
4.
Click Submit. The Search Subject Data page appears.
5.
To restrict your SNP subject data search to a specific population or analysis group, select one from the drop-down menus. An abstract of the analysis appears in the Analysis Group Description text block.

If you do not select a population or an analysis group, the search returns data from all populations and analysis groups.
6.
You can further refine your search by specifying additional information using the search criteria described in the following table.
Search Criteria
Explanation
Gender
Select gender from the drop-down menu
Age
Select upper and lower range from the drop-down menus
Case control status
Study participant association with disease status
Family History
Select yes or no from the drop-down menu
7.
Click Submit. The SNP Subject Data Report appears. See Understanding the SNP Subject Data Report for more information.
To clear your selections and start a new search, click Reset.
